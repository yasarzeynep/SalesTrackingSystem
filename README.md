# Satış Takip Sistemi

Bu proje, satışları takip etmek için kullanılan bir web uygulamasıdır.

## Kurulum

1. Depoyu klonlayın: `git clone https://github.com/kullanici_adi/SatisTakipSistemi.git`
2. Proje dizinine gidin: `cd SatisTakipSistemi`
3. Bağımlılıkları yükleyin: `npm install`

## Katkıda Bulunma

Katkılarınızı bekliyoruz! Lütfen depoyu çatallayın (fork) ve bir pull request gönderin.
